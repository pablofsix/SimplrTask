# **App Name**: ProjectFlow

## Core Features:

- Project Creation: Allow users to create new projects with a unique name.
- Task Input Form: Provide a form for entering tasks/descriptions, limited to 150 characters, for each project.
- Task Listing: List tasks under their respective projects as they are added via the input form.
- Modification History: Track and store a history of modifications for each task within a project. Implemented using Cloud SQL database
- Summarization of current tasks.: Provide generative AI powered summaries of current tasks.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey organization and focus.
- Background color: Very light blue (#E8EAF6) for a calm, uncluttered workspace.
- Accent color: Violet (#9575CD) to highlight important actions and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look, used in both headlines and body text.
- Use clear and intuitive icons for tasks and project actions.
- Clean and structured layout for easy task and project management.
- Subtle animations to indicate actions and changes, providing user feedback.
