## Project Overview

ProjectFlow nace de una necesidad práctica: gestionar tareas y estados durante reuniones técnicas de forma simple e intuitiva. El proyecto representa un ciclo completo de ingeniería: identificar un problema, diseñar una arquitectura funcional, implementarla con herramientas modernas y desplegarla.

## What I Did

- **Problem Definition**: Identificó la necesidad y definió requisitos funcionales basado en experiencia en gestión de proyectos
- **Architecture Design**: Diseñó la estructura de datos, lógica de estado y modelo de persistencia
- **Initial Prototyping**: Usó Firebase Studio para validar conceptos rápidamente
- **Implementation**: Desarrolló la aplicación usando VS Code + GitHub Copilot, implementando features como:
  - Gestión multi-proyecto
  - Rastreo de cambios y auditoría
  - Sincronización entre pestañas
  - Almacenamiento local persistente
- **Production Deployment**: Configuró CI/CD automático con GitHub Pages + GitHub Actions
- **Documentation**: Creó guías para usuarios y desarrolladores

## Technologies Used

- **Prototyping**: Firebase Studio
- **Development**: VS Code, GitHub Copilot
- **Frontend**: Next.js 15, Radix UI, Tailwind CSS
- **Storage**: Browser localStorage
- **Deployment**: GitHub Pages, GitHub Actions
- **AI**: Genkit + Google Gemini 2.5 Flash

## Result

Una aplicación funcional, mantenible y en uso que demuestra cómo convertir un problema real en una solución práctica usando herramientas modernas.
